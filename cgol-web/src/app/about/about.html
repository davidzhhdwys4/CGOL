About the app

See Wikipedia article for more information. This app is served from my github repo.

Why make this?

This app is a combined tutorial of the following tools:
  • Angular
  • Angular Material
  • Rust and WebAssembly
  • IndexedDB via Dexie.js
  • Github Actions
The app uses the angular framework and angular material components for the frontend. The backend logic of the game is implemented in Rust and compiled to WebAssembly, and the game states are saved in the browser's IndexedDB using Dexie.js.

There is an angular service worker that enables offline support. Try turning on airplane mode and refreshing the page - the app will still load! In Edge and Chrome, you can also install the app as a PWA (Progressive Web App) by clicking the install button in the address bar.
